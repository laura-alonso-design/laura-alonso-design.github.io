UX/UI

User needs

The user needs a easy way to understand all the options that he has before acquiring a Rayvolt e-bike.

Strategy

The user target of this website is a man older than 45. This means that he needs an easy way to acquires a Rayvolt e-bike which is a product with a lot of custom options, technical configuration as well as accessories. The UI respects the warm vintage look of the product and the color are conservative, except the interactive elements which with an intense color lead the user throgh the interface.




 

Main interactive color

The orange color leads the user to keep easily the steps of the product configuration.

Vintage look and feel

Appropriate aesthetics for the user target.

Easy to understand

The interaction is organized and clear, so the user has no problem understanding the product options.
